# 辩论队数据统计系统

这是一个轻量级单页原型，核心定位是“外部超链接挂载的赛事资料库”：比赛录像、网盘资料、文档文件都通过 URL 接入，本地不保存文件内容。

## 当前功能

- 数据概览：总场次数、胜场数、赛季总胜率、覆盖赛事数
- 筛选控件区：预留队员、年份、赛事筛选入口
- 赛事列表：展示赛事信息、辩题、持方、赛果、最佳辩手、出场阵容
- 外部链接：每场比赛提供“录像”和“资料”按钮

## 数据结构

当前页面数据写在 `src/matches.json` 中。后续可以把这个静态 JSON 文件替换为 Notion API、Airtable API 或其他云表格 API 返回的数据。

单场比赛字段：

```json
{
  "id": "唯一标识符",
  "date": "YYYY-MM-DD",
  "tournament": "赛事名称",
  "motion": "辩题",
  "side": "正方/反方",
  "result": "胜/负/平",
  "bestDebater": "本场最佳辩手姓名",
  "lineup": ["一辩", "二辩", "三辩", "四辩/结辩"],
  "videoUrl": "比赛录像外部链接",
  "docUrl": "赛事资料外部链接"
}
```
