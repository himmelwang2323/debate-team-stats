# 外交学院辩论队数据统计

## 当前功能

- 数据概览：总场次数、胜场数、赛季总胜率、覆盖赛事数
- 数据检索：按姓名、日期、赛事关键词即时筛选
- 赛事列表：展示赛事信息、辩题、持方、赛果、票型、最佳辩手、出场阵容
- 外部链接：每场比赛提供“录像”和“资料”按钮

## 数据结构

当前页面数据写在 `src/matches.json` 中。后续可以把这个静态 JSON 文件替换为 Notion API、Airtable API 或其他云表格 API 返回的数据。

单场比赛字段：

```json
{
  "id": "唯一标识符",
  "date": "YYYY-MM-DD",
  "tournament": "赛事名称",
  "motion": "辩题",
  "side": "正方/反方",
  "result": "胜/负/平",
  "voteSplit": "票型，如 X：X",
  "bestDebater": "本场最佳辩手姓名",
  "lineup": ["一辩", "二辩", "三辩", "四辩/结辩"],
  "videoUrl": "比赛录像外部链接",
  "docUrl": "赛事资料外部链接"
}
```
